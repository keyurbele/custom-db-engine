# Contributing to Custom DB Engine

Thank you for checking out this project! This is an educational storage engine designed to explore low-level database internals. Contributions, bug reports, and optimizations are welcome.

## How to Contribute

1. **Fork the Repository:** Create a personal copy of the repository on your GitHub account.
2. **Clone Locally:** Download the project files to your machine.
   ```bash
   git clone [https://github.com/YOUR_USERNAME/custom-db-engine.git](https://github.com/YOUR_USERNAME/custom-db-engine.git)
   cd custom-db-engine
